# Introduction to Machine Learning

## What is this?

This repository holds the online pages of the course Introduction to Machine Learning.

It is designed for the Master's course [AI For Science](https://ai.aims.ac.za/), delivered at the [African Institute for Mathematical Sciences (AIMS)](https://nexteinstein.org) and made possible through a partnership with [Google DeepMind](https://www.deepmind.com/).

The course uses [Jupyter-Book](https://jupyterbook.org/en/stable/intro.html) as a wrapper for displaying lecture content and exercises.

Programming tutorials and assignments will be done in Python using [Jupyter Notebooks](https://jupyter.org/).

## Calendar

This course is being delivered at AIMS South Africa in September 2023. 

It will be delivered in January 2024 at AIMS Senegal and in February 2024 at AIMS Cameroon.
