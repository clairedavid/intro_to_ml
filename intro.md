# Introduction to Machine Learning
&nbsp;
 
  __<Big>Learn the math behind machine learning!</Big>__  


Learn how to code a machine learning algorithm in python, seeing the math at play.<br>

Learn how to diagnose and assess the performance of your algorithm.

Learn how to learn 😉, autonomously and ethically ⚖️, about Artificial Intelligence.

Learn how to apply machine learning to beautiful problems in your favourite science field!
